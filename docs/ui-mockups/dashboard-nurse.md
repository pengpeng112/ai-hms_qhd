## 取舍说明

- **为什么放这 4 张**：护士核心关注"今天哪些床有人"→"要执行什么"→"谁还没评估"→"设备是否正常"
- **为什么不放患者增长/统计**：这些是管理层关注指标，护士日常工作不需要
- **在线设备统计**：若 `getDevices` 不支持 status 过滤，降级为全量设备列表 + 前端过滤
